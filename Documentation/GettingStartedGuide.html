Minimum Requirements

iOS 4 is minimum supported version by this SDK. 

Install the SDK

Instructions for Xcode 4:

  1. Open SocializeSDK.xcodeproj
  2. Select SocializeSDK as the target.
  3. From the Product menu, select Build For -> Build For Archiving

Look in the output/ directory.

There you should static library along with all the header

Copy these files to somewhere in your project directory and use them.

If you want to build the debug version instead, just select Product -> Build, and look in the output/ directory.

Authentication

Where delegate is a pointer to user object that implements SocializeServiceDelegate protocol or nil.

Step 1:- Instantiate

// Allocate memory for the instance

Socialize* socialize = [[Socialize alloc] initWithDelegate:nil];

// invoke the call

[socialize authenticateWithApiKey:@”ApiKey” apiSecret:@”ApiSecret” delegate:self];

Step 2:- Implement the SocializeAuthenticationDelegate

-(void)didAuthenticate{

     DLog(@”Authenticated”);

}

-(void)didNotAuthenticate:(NSError*)error{

    DLog(@"%@", error);

}

Like an Entity

Step 1: For liking

// Allocate memory for the instance

Socialize* socialize = [[Socialize alloc] initWithDelegate:self];

// invoke the call

[socialize likeEntityWithKey:@”www.bbc.co.uk”];

Step 2: Implementing the delegate

//if the like fails this is called

-(void)service:(SocializeService*)service didFail:(NSError*)error{

}

// if the like is created this is called

-(void)service:(SocializeService*)service didCreateWithElements:(NSArray*)dataArray andErrorList:(id)errorList{

          if ([[dataArray objectAtIndex:0]  

                              conformsToProtocol:@protocol(SocializeLike) ]){

           //_like is object level instance of type <SocializeLike>

            _like = [dataArray objectAtIndex:0];

            [_like  retain];

        }

}

Unlike an Entity

Step 1: For unliking

// Allocate memory for the instance

Socialize* socialize = [[Socialize alloc] initWithDelegate:self];

// invoke the call where _like is of Type id <SocializeLike> which was previously returned // when the entity was liked

[socialize [self.socialize unlikeEntity:_like];

Step 2: Implementing the delegate

//if the unlike fails this is called

-(void)service:(SocializeService*)service didFail:(NSError*)error{

}

// if the entity is unliked this is called

-(void)service:(SocializeService*)service didDelete:(id<SocializeObject>)object{

}
